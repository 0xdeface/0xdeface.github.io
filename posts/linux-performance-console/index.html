Must have для удобного работы. ZSH и стеройды.

Posted on Feb 20, 2022

Рекомендую как стандарт работы в консоли linux.

  1. zsh - прокачанный bash, полностью совместимый. Улучшенный автокомплит, ui и многое другое.
pacman -S zsh
apt install zsh
yum install zsh
  1. oh-my-zsh - коллекция плагинов и улучшений для вашего zsh.
  2. любимая тема оформления для zsh “powerlevel10k/powerlevel10k”
  3. fzf - fuzzy search. поиск по частям слова. [url]https://github.com/junegunn/fzf[/url]
  4. fzf-tab - автокомплит с использованием fzf
