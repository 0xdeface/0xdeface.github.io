ОШИБКА ПРИ УСТАНОВКЕ (ОБНОВЛЕНИИ) ПАКЕТОВ ЧЕРЕЗ PYCHARM

Собственно делюсь маленьким патчем чтобы старенький PyCharm продолжал исправно работать с новым pip нужно поправить хелпер, путь к нему есть в описании к ошибке. У меня он /pycharm-2018.1/helpers/packaging_tool.py ~~~~python

pycharm-2018.1/helpers/packaging_tool.py

def do_install(pkgs): try: import pip except ImportError: error_no_pip() if hasattr(pip,‘main’): return pip.main([‘install’] + pkgs) else: from pip._internal import main return main([‘install’] + pkgs) ~~~~
